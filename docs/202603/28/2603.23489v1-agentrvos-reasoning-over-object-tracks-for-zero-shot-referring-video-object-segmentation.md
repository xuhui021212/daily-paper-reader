## 摘要
指代视频物体分割 (RVOS) 旨在根据自然语言查询在整个视频中分割目标物体。该任务的免训练方法通常遵循一个通用流程：多模态大语言模型 (MLLM) 选择关键帧，在这些帧中定位被指代的物体，然后由视频分割模型传播结果。虽然这种设计很直观，但它要求 MLLM 在获得任何物体级证据之前做出时间决策，从而限制了推理质量和时空覆盖范围。为了克服这一问题，我们提出了 AgentRVOS，这是一种基于 SAM3 和 MLLM 互补优势构建的免训练智能体流水线。给定从查询中提取的概念，SAM3 通过生成的掩码轨迹在整个时空范围内提供可靠的感知。随后，MLLM 通过对这些物体级证据进行基于查询关联的推理来识别目标，并在 SAM3 时间存在信息的指导下进行迭代剪枝。大量实验表明，AgentRVOS 在多个基准测试中达到了免训练方法的最先进性能，并在不同的 MLLM 骨干网络上表现出一致的结果。我们的项目页面见：https://cvlab-kaist.github.io/AgentRVOS/。

## Abstract
Referring Video Object Segmentation (RVOS) aims to segment a target object throughout a video given a natural language query. Training-free methods for this task follow a common pipeline: a MLLM selects keyframes, grounds the referred object within those frames, and a video segmentation model propagates the results. While intuitive, this design asks the MLLM to make temporal decisions before any object-level evidence is available, limiting both reasoning quality and spatio-temporal coverage. To overcome this, we propose AgentRVOS, a training-free agentic pipeline built on the complementary strengths of SAM3 and a MLLM. Given a concept derived from the query, SAM3 provides reliable perception over the full spatio-temporal extent through generated mask tracks. The MLLM then identifies the target through query-grounded reasoning over this object-level evidence, iteratively pruning guided by SAM3's temporal existence information. Extensive experiments show that AgentRVOS achieves state-of-the-art performance among training-free methods across multiple benchmarks, with consistent results across diverse MLLM backbones. Our project page is available at: https://cvlab-kaist.github.io/AgentRVOS/.

#### 1. 核心问题与研究动机
*   **核心问题**：指代视频物体分割（RVOS）要求模型根据自然语言描述在视频中精准分割目标。现有的免训练（Training-free）方法通常让多模态大模型（MLLM）先选择关键帧再进行定位。
*   **研究动机**：
    *   **时空覆盖不足**：由于 Token 限制，MLLM 只能处理稀疏采样帧，容易漏掉出现时间短或非显著的物体。
    *   **决策顺序不合理**：现有方法要求 MLLM 在没有任何物体级证据的情况下预先做出时间决策（选帧），限制了推理质量。
    *   **感知与推理失衡**：MLLM 擅长复杂逻辑推理但感知精度有限；而分割模型（如 SAM2/SAM3）感知强但无法理解复杂指令。

#### 2. 方法论：核心思想与算法流程
AgentRVOS 提出了一种“感知引导推理”的智能体流水线，将任务分为两个阶段：
*   **阶段一：候选掩码轨迹生成（Candidate Generation）**
    *   **概念提取**：MLLM 将复杂查询分解为 SAM3 可理解的“核心概念”（如：猫）和“宽泛概念”（如：动物）。
    *   **全时空感知**：利用 SAM3 处理视频所有帧，生成候选物体轨迹（Mask Tracks）。若未检测到物体，MLLM 会迭代修正概念进行重试，确保高召回率。
*   **阶段二：迭代时空剪枝（Iterative Spatio-temporal Pruning）**
    *   **物体级推理**：MLLM 观察带有颜色标注（Visual Prompting）的候选轨迹视频。
    *   **三分类决策**：MLLM 将候选物体标记为“接受（Accepted）”、“拒绝（Rejected）”或“不确定（Uncertain）”。
    *   **时空收缩**：在下一轮迭代中，系统自动将时间范围缩小至仅包含“不确定”物体的帧，并剔除已拒绝的干扰项。
    *   **外观工具（可选）**：针对颜色相关的查询，调用外观工具提取物体的原始视觉特征，弥补掩码覆盖造成的视觉损失。

#### 6. 主要结论与发现

*   **感知与推理的互补性**：证明了将“物体检测/跟踪”外包给感知模型（SAM3），让 MLLM 专注于“物体级证据推理”是解决零样本 RVOS 的最优路径。

#### 7. 优点
*   **高召回率**：通过 SAM3 全帧扫描和迭代概念修正，极大地减少了漏检。
*   **推理效率高**：迭代剪枝机制让 MLLM 能够逐步聚焦于难例，避免了在冗余信息上的计算浪费。
*   **通用性强**：作为免训练方案，可以无缝切换最新的 MLLM 骨干，具有极强的扩展性。

#### 8. 不足与局限
*   **计算开销**：虽然免训练，但多次调用 MLLM 进行迭代推理以及 SAM3 处理全视频帧，在实时性要求高的场景下可能存在延迟。
*   **依赖感知模型**：如果 SAM3 在第一阶段完全无法提取出目标物体的任何轨迹（即使经过概念修正），后续推理将无法挽回。
*   **长视频挑战**：对于极长视频，虽然有剪枝机制，但初始的候选池可能非常庞大，对 MLLM 的上下文处理能力仍有一定要求。
