## 摘要
视觉智能体记忆在流式视频理解中展现出了显著的有效性。然而，为视频存储此类记忆会产生巨大的内存开销，导致存储和计算成本高昂。为了解决这一问题，我们提出了 StreamMeCo，一个高效的流式智能体记忆压缩框架。具体而言，基于记忆图的连通性，StreamMeCo 为孤立节点引入了无边极大极小采样（edge-free minmax sampling），并为连通节点引入了边缘感知权重剪枝（edge-aware weight pruning），在保持准确性的同时剔除冗余记忆节点。此外，我们引入了一种时间衰减记忆检索机制，以进一步消除由记忆压缩引起的性能下降。在三个具有挑战性的基准数据集（M3-Bench-robot、M3-Bench-web 和 Video-MME-Long）上的广泛实验表明，在 70% 的记忆图压缩率下，StreamMeCo 实现了 1.87 倍的记忆检索加速，同时平均准确率提升了 1.0%。我们的代码可在 https://github.com/Celina-love-sweet/StreamMeCo 获取。

## Abstract
Vision agent memory has shown remarkable effectiveness in streaming video understanding. However, storing such memory for videos incurs substantial memory overhead, leading to high costs in both storage and computation. To address this issue, we propose StreamMeCo, an efficient Stream Agent Memory Compression framework. Specifically, based on the connectivity of the memory graph, StreamMeCo introduces edge-free minmax sampling for the isolated nodes and an edge-aware weight pruning for connected nodes, evicting the redundant memory nodes while maintaining the accuracy. In addition, we introduce a time-decay memory retrieval mechanism to further eliminate the performance degradation caused by memory compression. Extensive experiments on three challenging benchmark datasets (M3-Bench-robot, M3-Bench-web and Video-MME-Long) demonstrate that under 70% memory graph compression, StreamMeCo achieves a 1.87* speedup in memory retrieval while delivering an average accuracy improvement of 1.0%. Our code is available at https://github.com/Celina-love-sweet/StreamMeCo.

随着流式视频理解（如直播、实时监控、自动驾驶）需求的增长，传统的离线视频处理方法已不再适用。现有的视觉智能体（Vision Agent）方法（如 M3-Agent）通过构建**记忆图（Memory Graph）**来存储文本、人脸和声音节点，以解决长视频中的实体一致性和长期依赖问题。然而，随着视频时长增加，记忆图规模呈爆炸式增长，导致存储开销巨大且检索延迟极高，严重阻碍了实时响应。本文旨在提出一种高效的记忆压缩框架，在大幅降低内存占用的同时，保持甚至提升视频理解的准确率。

### 2. 方法论：核心思想与关键技术
StreamMeCo 提出了一个无需训练的长期智能体记忆压缩框架，核心包含双分支压缩策略和一种启发自人类记忆的检索机制：

*   **孤立节点压缩：无边极大极小采样 (EMsampling)**
    *   针对不与特定实体（人脸/声音）连接的孤立文本节点，首先使用 **Spherical KMeans** 算法根据语义嵌入进行聚类。
    *   在每个簇内，采用**极大极小采样策略**：先选中心点，再迭代选择与已选集合距离最远的节点，确保保留的节点具有最强的语义代表性和多样性。
*   **连接节点压缩：边感知权重剪枝 (EWpruning)**
    *   针对与人脸或声音节点相连的文本节点，构建**实体重要性评分**（基于边权重）和**嵌入相似度评分**。
    *   通过加权融合这两个评分得到最终的“融合得分”，优先保留对实体识别关键且语义独特的节点，剪掉冗余信息。
*   **时间衰减记忆检索 (TMR)**
    *   模拟人类大脑的遗忘曲线，引入**非线性时间衰减函数**（指数衰减）。
    *   在检索阶段，动态分配不同时间段的检索配额，确保模型在关注全局信息的同时，能自适应地优先检索近期关键信息。
