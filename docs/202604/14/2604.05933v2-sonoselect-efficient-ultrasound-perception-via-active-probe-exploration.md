## 摘要
超声感知通常需要通过移动探头获取多个扫描切面，以减少诊断歧义、缓解声学遮挡并提高解剖覆盖范围。然而，并非所有探头切面都具有同等的信息量。穷举式地获取大量切面会引入显著的冗余，增加扫描和处理成本。为解决这一问题，我们定义了超声主动切面探索任务，并提出了 SonoSelect，这是一种专门针对超声的方法，能够根据当前观测结果自适应地引导探头移动。具体而言，我们将超声主动切面探索建模为一个序列决策问题。每个新的二维（2D）超声切面都会融合到观测解剖结构的 3D 空间记忆中，从而引导下一个探头位置。在此框架基础上，我们提出了一个超声专用目标函数，该函数倾向于选择具有更大器官覆盖范围、更低重建不确定性以及更少冗余扫描的探头移动路径。在超声模拟器上的实验表明，SonoSelect 仅使用 N 个切面中的 2 个即可实现极具前景的多切面器官分类准确率。此外，对于更具挑战性的肾囊肿检测任务，它实现了 54.56% 的肾脏覆盖率和 35.13% 的囊肿覆盖率，且短轨迹始终以目标囊肿为中心。

## Abstract
Ultrasound perception typically requires multiple scan views through probe movement to reduce diagnostic ambiguity, mitigate acoustic occlusions, and improve anatomical coverage. However, not all probe views are equally informative. Exhaustively acquiring a large number of views can introduce substantial redundancy, increase scanning and processing costs. To address this, we define an active view exploration task for ultrasound and propose SonoSelect, an ultrasound-specific method that adaptively guides probe movement based on current observations. Specifically, we cast ultrasound active view exploration as a sequential decision-making problem. Each new 2D ultrasound view is fused into a 3D spatial memory of the observed anatomy, which guides the next probe position. On top of this formulation, we propose an ultrasound-specific objective that favors probe movements with greater organ coverage, lower reconstruction uncertainty, and less redundant scanning. Experiments on the ultrasound simulator show that SonoSelect achieves promising multi-view organ classification accuracy using only 2 out of N views. Furthermore, for a more difficult kidney cyst detection task, it reaches 54.56% kidney coverage and 35.13% cyst coverage, with short trajectories consistently centered on the target cyst.

超声成像高度依赖于扫描视角。在临床实践中，医生需要不断移动探头以减少诊断歧义、克服声学遮挡并扩大解剖覆盖范围。然而，传统的扫描方式存在两个主要痛点：
*   **冗余性与高成本**：穷举式扫描会产生大量重复信息，增加数据处理负担和扫描时间。
*   **操作者依赖性**：视角选择依赖医生经验，难以标准化。
论文提出了**主动视角探索（Active View Exploration）**任务，旨在通过算法自适应地引导探头移动，在有限的扫描预算内最大化诊断信息增益。

#### 2. 方法论：核心思想与关键技术
SonoSelect 将超声扫描建模为一个**部分可观测的序列决策问题**，核心思想是利用“空间记忆”指导“未来探索”。
*   **3D 空间记忆（Spatial Memory）**：系统维护一个 3D 概率体素网格 $\hat{V}_t$。每获取一帧 2D 超声图像，就利用已知的探头位姿将其投影并融合进网格中，通过贝叶斯融合更新体素占据概率，从而记录已观测区域并识别不确定区域。
*   **分层策略网络**：
    *   **视角选择模块（View Selection）**：将探头周围区域划分为多个扇区（Sectors），利用 Q-Learning 评估每个扇区的潜在奖励，决定“去哪儿扫描”（长程路由）。
    *   **动作精细化模块（Action Refinement）**：基于 PPO 算法，将选定的扇区目标转化为具体的连续运动指令（短程控制）。
*   **残差融合（Residual Fusion）**：最终动作由扇区引导向量与策略网络输出的精细增量加权融合而成，并随训练过程动态调整权重。
*   **多目标奖励函数**：奖励项包含：1) 目标解剖结构的覆盖率增益；2) 区域信息熵的减少（降低不确定性）；3) 路径惩罚（减少无用移动）。

#### 3. 实验设计
实验分为两个阶段进行验证：
*   **阶段一：多视角分类（离散设置）**

    *   **对比方法**：数据集/实例级 Oracle、随机选择、MVSelect 等。

#### 6. 主要结论与发现
*   **高效性**：在分类任务中，仅需 2 个自适应选择的视角即可达到或超过全量视角的准确率。

*   **目标聚焦**：定性分析显示，SonoSelect 的扫描轨迹能紧密围绕目标器官，有效减少了在非目标区域的扫描预算浪费。

#### 7. 优点（亮点）
*   **分层架构设计**：将复杂的连续控制分解为“离散路由 + 连续微调”，降低了学习难度，提高了决策的逻辑性。
*   **空间记忆机制**：引入 3D 概率图作为状态表示，解决了超声扫描中的部分可观测性问题，使智能体具备“全局观”。
*   **临床潜力**：该方法不仅适用于机器人自动化扫描，其视角建议也可作为辅助决策工具支持超声医生的手动操作。

#### 8. 不足与局限
*   **模拟环境限制**：实验完全在基于 CT 重建的超声模拟器中进行，未考虑真实超声中的复杂伪影（如混响、侧壁失影）以及人体呼吸运动带来的动态干扰。
