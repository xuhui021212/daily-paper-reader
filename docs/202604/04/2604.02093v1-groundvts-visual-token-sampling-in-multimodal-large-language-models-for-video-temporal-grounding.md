## 摘要
视频时序定位（VTG）是视频理解中的一项关键任务，也是将视频大语言模型（Vid-LLMs）扩展到更广泛应用的核心能力。然而，现有的 Vid-LLMs 依赖均匀帧采样来提取视频信息，导致关键帧分布稀疏并丢失了重要的时序线索。为了解决这一局限性，我们提出了 Grounded Visual Token Sampling (GroundVTS)，这是一种专注于最具信息量的时序片段的 Vid-LLM 架构。GroundVTS 采用细粒度的查询引导机制，在将视觉 Token 输入 LLM 之前对其进行过滤，从而保留了必要的时空信息并维持了时序连贯性。此外，我们引入了一种渐进式优化策略，使 LLM 能够有效适应视觉特征的非均匀分布，增强其建模时序依赖关系和实现精确视频定位的能力。我们在三个标准的 VTG 基准测试上对 GroundVTS 进行了全面评估，结果显示其优于现有方法，在时刻检索（moment retrieval）的 mIoU 上提升了 7.7 个百分点，在亮点检测（highlight detection）的 mAP 上提升了 12.0 个百分点。代码可在 https://github.com/Florence365/GroundVTS 获取。

## Abstract
Video temporal grounding (VTG) is a critical task in video understanding and a key capability for extending video large language models (Vid-LLMs) to broader applications. However, existing Vid-LLMs rely on uniform frame sampling to extract video information, resulting in a sparse distribution of key frames and the loss of crucial temporal cues. To address this limitation, we propose Grounded Visual Token Sampling (GroundVTS), a Vid-LLM architecture that focuses on the most informative temporal segments. GroundVTS employs a fine-grained, query-guided mechanism to filter visual tokens before feeding them into the LLM, thereby preserving essential spatio-temporal information and maintaining temporal coherence. Futhermore, we introduce a progressive optimization strategy that enables the LLM to effectively adapt to the non-uniform distribution of visual features, enhancing its ability to model temporal dependencies and achieve precise video localization. We comprehensively evaluate GroundVTS on three standard VTG benchmarks, where it outperforms existing methods, achieving a 7.7-point improvement in mIoU for moment retrieval and 12.0-point improvement in mAP for highlight detection. Code is available at https://github.com/Florence365/GroundVTS.

### 2. 方法论：核心思想与关键技术
GroundVTS 提出了一种**查询引导的视觉 Token 采样（VTS）机制**，其核心流程如下：
*   **查询引导的 Token 评分**：在视觉编码器和多模态投影层之后，利用文本查询的嵌入向量与视觉 Token 进行相似度计算。通过线性投影将两者映射到低维空间，计算点积并经过 Softmax 归一化，得到每个视觉 Token 的相关性得分。
*   **可微 Top-K 选择**：为了保留最具信息量的 Token，模型根据得分选择前 $K$ 个 Token（采样率 $\rho$ 通常设为 0.5）。为了解决 Top-K 操作不可微的问题，引入了 **Straight-Through Estimator (STE)** 和 **Gumbel-Softmax 弛豫**，确保采样过程可以在端到端训练中传递梯度。
*   **保持时序连贯性**：采样后的 Token 保留其原始的相对位置编码（Positional Encoding），确保非均匀分布的 Token 在输入 LLM 时仍能维持正确的时序逻辑。
*   **渐进式优化策略**：
    1.  **VTS 预热（Warm-up）**：仅训练采样模块，使其学习初步的相关性估计。

    3.  **定位微调（Grounding FT）**：在专门构建的 Grounding-FT 数据集上进行最终的定位任务强化训练。

*   **细粒度采样**：不同于以往的“帧级”筛选，GroundVTS 实现了“Token 级”筛选，能够剔除帧内无关的背景信息。

*   **早期过滤风险**：采样发生在多模态融合的早期阶段。虽然高效，但可能在深层语义交互之前就丢失了一些潜在有用的视觉细节（论文在讨论部分也提到了这一点）。

*   **任务局限**：主要针对时序定位任务优化，虽然在通用 VQA 上表现稳定，但在需要极高空间细节（如细微物体识别）的任务中，50% 的采样率可能存在风险。
