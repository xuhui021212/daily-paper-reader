## 摘要
视频大语言模型 (VLLMs) 展现出强大的视频理解能力，但由于冗余的视觉 Token，面临效率低下的问题。现有的剪枝方法主要针对帧内空间冗余，或在 LLM 内部进行具有浅层开销的剪枝，导致时空缩减效果不佳，且未能充分利用长上下文的可压缩性。这些方法通常会丢弃来自合并或剪枝 Token 的细微但具有信息量的上下文。在本文中，我们提出了一种新视角，通过局部-全局最优传输 (AOT) 精细化构建帧内和帧间的 Token 锚点，从而全面聚合信息丰富的上下文。具体而言，我们首先在注意力机制的引导下，在每帧内建立局部和全局感知的 Token 锚点，随后通过最优传输聚合来自被剪枝 Token 的信息上下文，构建帧内 Token 锚点。接着，基于时间视频片段，每个片段的第一帧将被视为关键帧锚点，通过最优传输整合来自连续帧的相似信息，同时保留独特的 Token 以表征时间动态，从而以无需训练的方式实现高效的 Token 缩减。广泛的评估表明，我们提出的 AOT 在主流视频 LLM 的各种短视频和长视频基准测试中均获得了具有竞争力的性能，在保持时间与视觉保真度的同时，显著提升了计算效率。

## Abstract
Video Large Language Models (VLLMs) demonstrate strong video understanding but suffer from inefficiency due to redundant visual tokens. Existing pruning primary targets intra-frame spatial redundancy or prunes inside the LLM with shallow-layer overhead, yielding suboptimal spatiotemporal reduction and underutilizing long-context compressibility. All of them often discard subtle yet informative context from merged or pruned tokens. In this paper, we propose a new perspective that elaborates token \textbf{A}nchors within intra-frame and inter-frame to comprehensively aggregate the informative contexts via local-global \textbf{O}ptimal \textbf{T}ransport (\textbf{AOT}). Specifically, we first establish local- and global-aware token anchors within each frame under the attention guidance, which then optimal transport aggregates the informative contexts from pruned tokens, constructing intra-frame token anchors. Then, building on the temporal frame clips, the first frame within each clip will be considered as the keyframe anchors to ensemble similar information from consecutive frames through optimal transport, while keeping distinct tokens to represent temporal dynamics, leading to efficient token reduction in a training-free manner. Extensive evaluations show that our proposed AOT obtains competitive performances across various short- and long-video benchmarks on leading video LLMs, obtaining substantial computational efficiency while preserving temporal and visual fidelity. ### 1. 核心问题与研究动机
*   **核心问题**：视频大语言模型（VLLM）在处理长视频时，视觉 Token 数量巨大（动辄数万甚至百万级），导致推理时的计算开销（FLOPs）和内存占用极高。
*   **研究动机**：
    *   **现有方法不足**：目前的 Token 缩减技术（如剪枝或合并）要么只关注帧内空间冗余，要么在 LLM 内部进行简单丢弃，往往会损失细微但关键的语义上下文，且未能充分利用视频的时空相关性。
    *   **信息流失**：简单的“丢弃”或“平均合并”无法有效保留非显著区域的有用信息。
    *   **目标**：开发一种**无需训练（Training-free）**的方法，在大幅减少 Token 数量的同时，通过优化手段将冗余信息“聚合”到保留的 Token 中。

### 2. 核心方法论：AOT (Local-Global Optimal Transport)
论文提出了基于**最优传输（Optimal Transport, OT）**的 Token 缩减框架，分为两个阶段：

*   **锚点建立（Anchors Establishment）**：
    *   **全局锚点**：利用视觉编码器的注意力分数（如 [CLS] 关注度），选取全局最重要的 Token。
    *   **局部锚点**：采用类似 AnyRes 的技术，在网格化窗口内选取局部关键 Token，确保空间多样性。
*   **阶段 I：帧内（Intra-Frame）聚合**：
    *   将选中的锚点视为“需求方”，未选中的 Token 视为“供应方”。
    *   利用 **Sinkhorn-Knopp 迭代**快速求解最优传输矩阵，根据传输计划（Transport Plan）将未选中 Token 的特征加权聚合到锚点中，从而实现“信息压缩而非简单丢弃”。
*   **阶段 II：帧间（Inter-Frame）聚合**：
    *   将视频划分为片段（Clip），以片段首帧作为时间锚点。
    *   通过 OT 衡量后续帧与锚点的相似度。相似信息被聚合进锚点，而具有显著动态变化的 Token 则被保留，以捕捉视频的时间演变。

### 6. 主要结论与发现
*   **高效性**：在仅保留 **10%** 视觉 Token 的情况下，AOT 依然能保持原始模型约 **97.6%** 的性能。
*   **计算节省**：计算开销（FLOPs）可缩减至原始模型的 **8.3%**。
*   **超越基准**：在某些长视频任务中，缩减后的模型性能甚至超过了全量 Token 的原始模型，这表明 AOT 有效过滤了视觉噪声，帮助 LLM 聚焦关键信息。
*   **长上下文优势**：通过 Token 缩减，模型能够处理比原始限制更多的视频帧（如 128 帧），突破了 LLM 上下文窗口的限制。

### 7. 优点与亮点
*   **信息无损化处理**：不同于传统的“剪枝”，AOT 通过最优传输理论实现了信息的“吸收”与“整合”，最大程度保留了背景或细微处的语义。
*   **即插即用**：无需任何微调或训练，可直接应用于现有的 VLLM 架构。
*   **时空协同优化**：同时解决了空间冗余和时间冗余，且在处理动态场景时具有较好的鲁棒性。

### 8. 不足与局限
*   **时间锚点启发式**：帧间聚合目前依赖于简单的片段划分和首帧锚点，对于极度复杂、镜头切换频繁的视频，这种启发式策略可能不够精准。
*   **超参数依赖**：虽然提供了默认值，但针对不同类型的视频（如静态监控 vs 快速运动体育视频），可能需要微调传输权重 $\lambda$ 以达到最优平衡。
*   **硬件加速适配**：虽然算法复杂度低，但在实际工程部署中，频繁的 Sinkhorn 迭代可能需要专门的算子优化才能在所有硬件上达到理论加速比。
