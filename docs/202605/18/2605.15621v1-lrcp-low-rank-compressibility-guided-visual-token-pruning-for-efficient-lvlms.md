## 摘要
大型视觉语言模型 (LVLMs) 实现了强大的多模态理解能力，但其推理成本随视觉 Token 数量的增加而迅速增长，尤其是在处理高分辨率图像和长视频时。现有的基于注意力的方法通过注意力分数估计 Token 的重要性，这可能会引入位置偏差；而基于表示的方法则根据特征关系或重建误差来减少视觉冗余，忽略了视觉 Token 集合的全局结构。在本文中，我们从低秩可压缩性的角度重新审视了视觉 Token 压缩。在不同的模型和数据集上，我们观察到视觉 Token 表示呈现出显著的低秩结构，其主导子空间即使在随机移除大部分 Token 后仍保持稳定。受此发现启发，我们提出了 LRCP，这是一个无需训练的压缩框架。它首先通过主成分分析 (PCA) 估计视觉 Token 的主导低秩子空间，然后根据每个 Token 在该子空间上的投影残差进行评分，保留那些难以被低秩背景解释的 Token。大量实验表明，LRCP 取得了优异的结果，在减少 88.9% Token 的情况下保留了 94.7% 的原始图像理解性能，并在减少 87.5% Token 的情况下保留了 97.8% 的平均视频理解准确率。

## Abstract
Large vision-language models (LVLMs) achieve strong multimodal understanding, but their inference cost grows rapidly with the number of visual tokens, especially for high-resolution images and long videos. Existing attention-based methods estimate token importance from attention scores, which may introduce positional bias, while representation-based methods reduce visual redundancy based on feature relations or reconstruction errors, overlooking the global structure of the visual token set. In this paper, we revisit visual token compression from the perspective of low-rank compressibility. Across models and datasets, we observe that visual token representations exhibit a pronounced low-rank structure, with a dominant subspace that remains stable even after a large fraction of tokens is randomly removed. Motivated by this finding, we propose LRCP, a training-free compression framework that first estimates the dominant low-rank subspace of visual tokens via PCA, and then scores each token by its projection residual onto this subspace, retaining tokens that are poorly explained by the low-rank background. Extensive experiments show that LRCP achieves superior results, preserving 94.7% of the original image-understanding performance with an 88.9% token reduction and 97.8% of the average video-understanding accuracy with an 87.5% token reduction.

*   **核心问题**：大型视觉语言模型（LVLM）在处理高分辨率图像和长视频时，视觉 Token 数量激增，导致推理时的计算开销（Prefilling 阶段）和内存占用巨大。
*   **研究动机**：
    *   **现有方法局限**：基于注意力（Attention-based）的方法易受位置偏差影响，且与 FlashAttention 等高效算子兼容性差；基于表示（Representation-based）的方法多关注局部关系，忽略了 Token 集合的全局结构。
    *   **新观察**：作者发现视觉 Token 在特征空间中呈现出显著且稳定的**低秩结构（Low-Rank Structure）**。即使随机移除大部分 Token，其主导子空间（Dominant Subspace）依然保持稳定。这意味着视觉冗余根植于这种全局共享的低秩背景中。

### 2. 方法论：LRCP 框架
LRCP（Low-Rank Compressibility Guided Pruning）是一种**无需训练**的即插即用剪枝方法，核心步骤如下：
1.  **低秩子空间构建**：对视觉编码器输出的 Token 矩阵进行主成分分析（PCA），提取前 $r$ 个主成分，构建能够代表“背景/共享信息”的主导子空间。
2.  **基于投影残差的评分**：计算每个 Token 在该子空间上的投影残差。残差越大，说明该 Token 越难以被低秩背景解释，携带的判别性/独特信息越多。
    *   公式逻辑：$s_i = \|x_i(I - P_r)\|^2_2$，其中 $P_r$ 是投影矩阵。
3.  **Token 剪枝与合并**：保留评分最高的 $K$ 个 Token。为了减少信息损失，将丢弃的 Token 根据余弦相似度合并到最接近的保留 Token 中（Token Merging）。
4.  **多阶段应用**：该方法可同时应用于视觉编码器输出端和 LLM 的中间层，且由于不依赖注意力权重，完全兼容 FlashAttention。

### 6. 主要结论与发现
*   **性能卓越**：在 LLaVA-v1.5 上，仅保留 64 个 Token（减少 88.9%）时，LRCP 仍能保持 94.7% 的原始性能，显著优于其他方法。

### 7. 优点与亮点
*   **理论新颖**：首次从全局低秩可压缩性的视角审视视觉 Token 冗余，而非单纯依赖局部相似性或注意力分数。
*   **高效兼容**：无需微调模型，计算开销极低（PCA 计算量远小于 LLM 前向传播），且支持现代硬件加速算子。

*   **硬件开销**：虽然 PCA 计算量小，但在极短序列上引入的额外计算可能会抵消部分加速效果（尽管在长序列上优势巨大）。
