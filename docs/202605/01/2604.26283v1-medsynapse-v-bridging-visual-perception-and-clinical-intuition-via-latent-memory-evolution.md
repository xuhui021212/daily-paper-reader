## 摘要
高精度医学诊断不仅依赖于静态影像特征，还依赖于专家在影像解读过程中即时调用的隐性诊断记忆。我们指出医学视觉语言模型（VLM）中由离散分词引起的一种根本性认知失配，这导致了量化损失、长程信息消散以及病例自适应专业知识的缺失。为了弥补这一差距，我们提出了 MedSynapse-V，这是一个潜空间诊断记忆演化框架，通过在模型的隐藏流中动态合成隐性诊断记忆，模拟临床医生的经验性调用。具体而言，该框架始于一种先验记忆元查询（Meta Query for Prior Memorization）机制，其中可学习的探针从解剖先验编码器中检索结构化先验，以生成压缩的隐性记忆。为确保临床保真度，我们引入了因果反事实精炼（CCR），利用强化学习和源自区域级特征掩码的反事实奖励来量化每条记忆的因果贡献，从而修剪冗余并将潜空间表示与诊断逻辑对齐。这一演化过程最终通过内在记忆转换（IMT）完成，这是一种特权-自主双分支范式，通过全词表散度对齐将教师分支的诊断模式内化到学生分支中。在多个数据集上的全面实证评估表明，通过将外部专业知识转化为内生参数，MedSynapse-V 在诊断准确性上显著优于现有的最先进方法，特别是思维链范式。

## Abstract
High-precision medical diagnosis relies not only on static imaging features but also on the implicit diagnostic memory experts instantly invoke during image interpretation. We pinpoint a fundamental cognitive misalignment in medical VLMs caused by discrete tokenization, leading to quantization loss, long-range information dissipation, and missing case-adaptive expertise. To bridge this gap, we propose ours, a framework for latent diagnostic memory evolution that simulates the experiential invocation of clinicians by dynamically synthesizing implicit diagnostic memories within the model's hidden stream. Specifically, it begins with a Meta Query for Prior Memorization mechanism, where learnable probes retrieve structured priors from an anatomical prior encoder to generate condensed implicit memories. To ensure clinical fidelity, we introduce Causal Counterfactual Refinement (CCR), which leverages reinforcement learning and counterfactual rewards derived from region-level feature masking to quantify the causal contribution of each memory, thereby pruning redundancies and aligning latent representations with diagnostic logic. This evolutionary process culminates in Intrinsic Memory Transition (IMT), a privileged-autonomous dual-branch paradigm that internalizes teacher-branch diagnostic patterns into the student-branch via full-vocabulary divergence alignment. Comprehensive empirical evaluations across multiple datasets demonstrate that ours, by transferring external expertise into endogenous parameters, significantly outperforms existing state-of-the-art methods, particularly chain-of-thought paradigms, in diagnostic accuracy.

### 1. 核心问题与整体含义（研究动机和背景）
论文指出，现有的医学视觉语言模型（VLM）在模拟专家临床直觉方面存在**认知失调**。
*   **离散标记的局限性**：传统模型依赖离散的文本 Token 进行推理（如思维链 CoT），这会导致病理特征（如病灶密度的微小变化）在量化过程中丢失信息，且长程推理链容易产生信息消散和幻觉。
*   **缺乏临床直觉**：资深医生往往依靠“隐式诊断记忆”进行快速模式识别，而非每一步都进行冗长的逻辑推导。
*   **研究目标**：构建一个能够模拟临床医生经验调用过程的框架，通过在潜空间（Latent Space）演化诊断记忆，桥接视觉感知与临床直觉。

### 2. 方法论：核心思想与关键技术
MedSynapse-V 提出了一个三阶段的潜记忆演化范式：

1.  **元查询先验记忆（MQPM）**：
    *   **核心思想**：从预训练的解剖编码器（如 MedSAM3）中提取结构化先验。
    *   **技术细节**：设计“诊断记忆采样器”，利用 $N$ 个可学习的元查询探针（Probes）通过交叉注意力机制从编码器特征中提取关键病理模式，将其转化为“隐式记忆向量”注入 VLM 的隐藏流中。
2.  **因果反事实精炼（CCR）**：
    *   **核心思想**：利用强化学习（RL）确保记忆向量与诊断逻辑具有因果相关性。
    *   **技术细节**：采用 GRPO 算法，引入**因果反事实奖励**。通过掩码（Masking）图像中的关键区域并观察模型预测的变化，量化每个记忆元素的因果贡献，从而修剪冗余信息并强化关键诊断特征。
3.  **内在记忆转换（IMT）**：
    *   **核心思想**：将对外部编码器的依赖转化为模型的内在能力。
    *   **技术细节**：采用“特权-自主”双分支蒸馏架构。教师分支使用外部编码器，学生分支仅依靠 VLM 自身的视觉特征生成记忆。通过全词表 Jensen-Shannon 散度（JSD）对齐，使模型在推理时无需外部编码器即可自主生成等效的潜空间记忆。

*   **数据集**：涵盖 7 个医学多模态基准，包括 VQA-RAD、SLAKE、PathVQA、PMC-VQA（闭口 VQA）、MMMU Health & Medicine（临床推理）、MedXpertQA-MM（专家级推理）和 GMAI-MMBench（多粒度评估）。

### 5. 实验数量与充分性
*   **实验规模**：在 7 个不同类型的基准数据集上进行了全面测试，涵盖了放射、病理、皮肤、眼底等 8 种影像模态。
*   **消融实验**：非常充分。作者针对训练阶段（MQPM/CCR/IMT）、奖励设计（是否包含因果奖励）、编码器选择、探针数量 $N$、注入位置等进行了详细的对比实验。
*   **客观性**：通过 t-SNE 可视化展示了记忆空间的演化过程，并提供了推理延迟与准确性的权衡分析，实验设计遵循了严格的零样本（Zero-shot）评估协议，具有较高的客观性和公平性。

### 6. 主要结论与发现
*   **性能领先**：MedSynapse-V 在所有基准测试中均优于现有的医学 VLM 和 CoT 范式，尤其在视觉接地（Visual Grounding）任务上优势明显。
*   **效率优势**：通过 IMT 机制，模型在推理时无需辅助编码器，推理延迟与标准 VLM 相当，远低于需要生成长文本 CoT 的模型。
*   **潜空间价值**：实验证明，在潜空间进行连续推理比在离散 Token 空间进行冗长推理更能减少幻觉，且因果反事实精炼能显著提升记忆的质量。

### 7. 优点（亮点）
*   **范式创新**：首次在医学领域提出将外部解剖先验通过 RL 和蒸馏转化为“内生潜记忆”的演化框架。
*   **因果对齐**：引入反事实干预来校准潜空间表示，这比单纯的文本偏好对齐更符合医学诊断的严谨性。
*   **推理高效**：成功解决了“性能提升依赖外部复杂模块”的矛盾，实现了轻量化推理。

### 8. 不足与局限
*   **罕见模态表现**：在训练数据较少的模态（如 OCT）上，记忆质量有所下降。
*   **多病灶处理**：当图像中存在多个相互竞争的病灶时，固定数量的记忆探针（$N=16$）可能会出现表示饱和，导致精度下降。
*   **细微特征区分**：对于极度相似的病例（如良性 vs 异型痣），潜空间记忆的判别粒度仍有提升空间。
